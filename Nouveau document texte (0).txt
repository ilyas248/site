Créer un site Web en 2 minutes, entièrement en HTML, et le publier instantanément est un défi, mais c'est tout à fait faisable. Voici un guide étape par étape pour y arriver :

1. Créez le fichier HTML
Ouvrez un éditeur de texte simple comme Notepad (Windows) ou TextEdit (Mac).
Collez le code HTML de base suivant :
html
Copier le code
<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Mon Site Web</title>
</head>
<body>
    <h1>Bienvenue sur mon site Web</h1>
    <p>Ceci est une page Web créée en 2 minutes.</p>
</body>
</html>
Enregistrez le fichier avec un nom comme index.html.

2. Publier le site Web gratuitement
GitHub Pages :
Allez sur GitHub et créez un compte si vous n'en avez pas.
Créez un nouveau dépôt (repository) nommé mon-site par exemple.
Téléchargez votre fichier index.html dans ce dépôt.
Allez dans les paramètres du dépôt, et sous la section "Pages", sélectionnez la branche principale (main) pour la source. Sauvegardez les paramètres.
Votre site sera disponible à une URL du type https://votre_nom_d_utilisateur.github.io/mon-site/.
Netlify Drop :
Rendez-vous sur Netlify Drop.
Glissez-déposez votre fichier index.html sur la page.
Netlify vous générera instantanément un lien unique pour votre site Web.

3. Accédez à votre site Web
Une fois ces étapes terminées, votre site sera en ligne. Vous pourrez le partager via l'URL fournie par GitHub Pages ou Netlify.

Cela vous permet de créer et de publier un site Web basique en moins de deux minutes.
